dirty-money
===========

Flash-powered visualization of industry contributions
to members of the US Congress. Created for RIT course 4002-331.

Design
------

The app uses a distinct color scheme invoking these hex values:
-	2E403B (blackish)
-	4F593F (greenish)
-	BFB68A (tan)
-	A69F7B (darker tan)
-	8E5440 (reddish)
